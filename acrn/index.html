ACRN Tinnitus Protocol

This is an attempt at implementing the Acoustic Coordinated Reset Neuromodulation tinnitus treatment protocol using this paper as a guide. At the moment, this page does not work in the latest versions of Firefox and Chrome. It does work in Safari.

First try to match your tinnitus frequency using the "Play Single Frequency" button and then start the protocol with the "Play ACRN" button. Individual frequency values and volume can be adjusted by clicking on them.
